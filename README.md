# fly-feel

为快速搭建开发环境所做的框架，内部涵盖大部分情况下的解决方案。

## fly-feel-core(完成)

定义最基础的返回对象和异常和常用依赖装配。

## fly-feel-define(进行中)

定义公共接口及对象

## fly-feel-dependencies

定义所有依赖版本的限定

## fly-feel-jdbc

定义各种类型的数据库中间件

## fly-feel-security

定义基础的账户权限关联关系

## fly-feel-security-oauth2

定义Oauth2版的管理，包括授权服务与资源服务

## fly-feel-task

定义基础的定时任务服务、

## fly-feel-test

定义测试模块

## fly-feel-verify

定义验证模块
