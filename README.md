# fly-feel

#### 介绍
为快速搭建开发环境所做的框架，内部涵盖大部分情况下的解决方案。

#### 软件架构
主要有以下特点：

- XSS防护
- 跨域处理
- 提供基础的业务支撑
- 完备的权限管理体系
- 基于Oauth2的权限服务与资源服务
- 验证码
- 文件管理
- 统一日志管理
